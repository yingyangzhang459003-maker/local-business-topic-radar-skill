# Output Template

Start every run with a concise run summary: Business Profile scope, research date, source layers checked, candidates rejected, and any evidence or compliance limits.

Then provide exactly ten numbered Topic Cards. Each card must contain every Topic Card v1 field, including its score breakdown, source links with publication dates, and `risk_notes`.

Close with a handoff instruction for `local-business-article-writer`, identifying the selected Topic Card and reminding the writer to preserve the structured fields and recheck freshness. This handoff is optional for the user and must not require the writer to run.
